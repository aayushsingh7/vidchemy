# VidChemy
Transforms a single influencer promo video into complete, Amazon-ready product listings — in minutes, not hours.

## Key Features
- **Video Analysis Pipeline**: Automatically processes uploaded influencer promo videos to detect products, extract key frames as listing images, and transcribe narration using speech-to-text.
- **NLP Content Generation**: Generates SEO-optimized product titles, bullet points, descriptions, and listing content using video dialogue and visual cues — tailored for high search rankings.
- **Smart Image Optimization**: Converts 16:9 video frames into clean 1:1 product images with automatic background removal, meeting marketplace image standards.
- **Multi-Language Support**: Supports both Hindi and English audio, making it ideal for India's diverse creator ecosystem.
- **7× Time Savings**: Reduces 5+ hours of manual listing creation to under 5 minutes per product.

## How It Works
- **Video Upload**: Upload a short-form promotional video (Reels/Shorts) to start the pipeline.
- **Smart Frame Extraction**: AI analyzes visual and audio context to detect the best product moment, then extracts a high-resolution frame using FFmpeg.
- **Audio Intelligence**: Speech-to-text and NLP extract product names, key features, and sentiment from the influencer's narration.
- **Auto-SEO Generation**: Generates Amazon/Flipkart-optimized titles, bullet points, descriptions, and high-ranking keywords.
- **Image Enhancement**: Automatically converts extracted 16:9 frames into marketplace-ready 1:1 product images with background removal.
- **Marketplace Ready**: Exports a structured data package containing the optimized product image and all listing metadata, ready for instant publishing.
